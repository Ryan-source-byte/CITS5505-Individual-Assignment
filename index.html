The Full-Stack Foundation

Mastering the web requires more than just syntax. It's about understanding the architecture that powers the modern internet. Let's dive into the core concepts.

01. HTML5: Structure & Semantics

HTML is the skeleton of the web. Modern standards emphasize Semantic HTML—using tags that describe their meaning. For example, instead of a generic <div>, we use the <nav> tag to define a navigation menu. This helps screen readers and search engines understand your site structure.

When you need to force a line break (like in a postal address or a poem), you use the <br> element. Unlike most tags, it doesn't need a closing tag!

The Importance of alt

Every image on the web should have an alt attribute. It provides alternative text if the image fails to load and is crucial for visually impaired users relying on screen readers. 

<!-- 1. Semantic Navigation -->
<nav>
  <a href="/">Home</a>
  <a href="/about">About</a>
</nav>

<!-- 2. Line Break Inside a Paragraph -->
<p>123 Web Dev Lane<br>Tech City, TX 75001</p>

<!-- 3. Accessible Image with Alt Text -->
<img src="profile.jpg" alt="Portrait of Ryan Gong">

02. CSS: Cascading Style Sheets

CSS (Cascading Style Sheets) is what makes the web beautiful. It controls layout, colors, and typography.

  • Backgrounds: To change the background color of an element, you use the background-color property.
  • Targeting Elements: To style specific elements inside others, use descendant selectors. For example, div p selects all <p> elements that are inside a <div>.
  • Flexbox Centering: Flexbox is a modern layout module. To center items horizontally along the main axis, you use justify-content: center;.
/* 1. Background Color */
body {
  background-color: #FFFFF0; 
}

/* 2. Descendant Selector */
div p {
  color: #6B5B73;
}

/* 3. Flexbox Centering */
.container {
  display: flex;
  justify-content: center; 
}

03. JavaScript: Logic & Data

Variables

Modern JavaScript introduced new ways to declare variables. Use the let keyword to declare a block-scoped variable that can be reassigned later. (Use const for values that never change, and avoid the outdated var).

DOM Selection

To manipulate the webpage, JS needs to find HTML elements. The most direct way to select an element with a specific ID (like id="demo") is using document.getElementById('demo').

JSON Parsing

When receiving data from a web server, it often comes as a JSON string. To convert this string into a usable JavaScript object, you use the JSON.parse() function. 

// 1. Variable Declaration
let score = 0;
score = 10; // 'let' allows reassignment
 
// 2. DOM Selection
const demoElement = document.getElementById('demo');
demoElement.textContent = "Hello World!";
 
// 3. JSON Parsing
const jsonString = '{"name": "Ryan", "role": "Developer"}';
const userObject = JSON.parse(jsonString);
console.log(userObject.name); // Outputs: Ryan
